# Outstanding Management App — Visual Design & Feature Reference

> **Purpose**: Feed this entire document into an image generation AI (NotebookLM, Midjourney, etc.) to generate a **cover image**, **Play Store screenshots**, and **promotional graphics** that accurately represent the app.

## App Identity

- **App Name**: Outstanding Manager
- **Tagline**: "AI-Powered Business Outstanding Tracker"
- **Target Users**: Small & medium business owners in India (textile traders, wholesale dealers, manufacturers)
- **Platform**: Android & iOS (Flutter cross-platform)
- **Visual Style**: Premium fintech aesthetic — clean, minimal, professional
- **Category**: Business / Finance / Accounting

## Typography

- **Font Family**: **Montserrat** (Google Fonts)
- **Headings**: Montserrat SemiBold (600), 20px
- **Body**: Montserrat Regular (400), 14px
- **Captions**: Montserrat Regular, 12px, grey
- **Financial Figures**: Montserrat Bold (700), 28px (for hero numbers)
- **Currency Format**: Indian Rupees `₹` with Indian comma grouping (e.g., ₹1,25,000.00)

## Navigation Structure

**Bottom Navigation Bar** with 4 tabs:
1. 🏠 **Home** (Dashboard)
2. 📈 **Sales** (Sales Outstanding invoices)
3. 📉 **Purchases** (Purchase Outstanding invoices)
4. ⚙️ **Settings** (Profile, Appearance, Legal, Account)

---

## Screen-by-Screen UI Description

### SCREEN 1: Dashboard (Home)

**Layout**: Single scrollable column with pull-to-refresh

**Components from top to bottom**:

1. **AppBar**: Title "Dashboard", transparent background, no elevation

2. **Net Outstanding Hero Card** (full width):
   - Background: Light green (`Colors.green.shade50`) if positive, light red (`Colors.red.shade50`) if negative
   - Left side: Large trending icon (↑ green or ↓ red), 36px
   - Right side: 
     - Label: "Net Outstanding" (grey, 13px)
     - Value: "₹3,45,000" (28px, bold, green or red)
     - Subtitle: "You will receive this" or "You owe this" (12px, grey)

3. **Two Metric Cards** (side by side, equal width):
   - **Left Card — "Sales Outstanding"**: Green down-arrow icon, value in bold 16px
   - **Right Card — "Purchase Outstanding"**: Red up-arrow icon, value in bold 16px
   - Both tappable (navigates to respective tabs)

4. **Smart Forecasting Section** (Cashflow Summary Widget):
   - Card with AI-generated summary text
   - Coverage percentage indicator
   - High-risk party warnings (if any)
   - Only shows when valid data is available

5. **Recent Activity Feed**:
   - Section header: "Recent Activity" (titleMedium, bold)
   - List of up to 10 items, each showing:
     - CircleAvatar with colored icon (green for sales/receipts, orange for purchases, red for payments)
     - Title: "Sale: Vasant Traders" or "Received from: Mehta & Sons"
     - Subtitle: "INV-001 • ₹50,000"
     - Trailing: Date "05 Mar"

6. **Floating Action Button** (bottom-right):
   - Extended FAB with sparkle icon (✨)
   - Label: "Pulse Assistant"
   - Background: `Colors.blue.shade700`
   - Text & icon: White
   - Opens the AI chat bottom sheet

### SCREEN 2: Pulse AI Assistant (Bottom Sheet)

**Opens from**: FAB on Dashboard

**Layout**: Modal bottom sheet with rounded top corners (20px radius)

**Components**:
- Chat-like interface
- Text input field at bottom: "Try: Received 50k from Vasant"
- AI processes natural language and shows parsed results:
  - Type detected (Sale / Purchase / Payment)
  - Party name matched
  - Amount extracted
  - Auto-allocated invoices
- Confirm/Cancel buttons

### SCREEN 3: Sales Outstanding

**Layout**: Scrollable list with search, sort, and export

**AppBar Actions**:
- Search icon → reveals search TextField
- Sort dropdown (Date, Amount, Party Name)
- Export menu (PDF, Excel)

**Body**:
- List of invoice cards, each containing:
  - Party name (bold, 16px)
  - Invoice number and date
  - Total amount and outstanding amount
  - Payment status badge: 
    - "Paid" — green chip
    - "Partial" — amber/orange chip  
    - "Unpaid" — red chip
  - Action buttons on tap: View Ledger, Record Payment, Payment History, Send Reminder, Delete

**FAB**: "+" button to add new Sales Record

### SCREEN 4: Purchases Outstanding

**Layout**: Mirror of Sales screen but for purchase invoices
- Same card structure, search, sort, and export capabilities
- No "Send Reminder" option (since these are bills TO pay, not to collect)

### SCREEN 5: Add Sales/Purchase Record

**Layout**: Form screen with fields:
- Party name (autocomplete from existing parties)
- Invoice number
- Invoice date (date picker)
- Due date (date picker)
- Total amount
- Notes (optional)
- Save button (coral red, full width)

### SCREEN 6: Record Payment

**Layout**: Form screen with:
- Party selector
- Payment amount
- Payment date
- Payment method (Cash, Bank Transfer, UPI, Cheque)
- Reference number (optional)
- Auto-allocated invoices list (FIFO allocation shown)
- Save button

### SCREEN 7: Party Ledger

**Layout**: Full transaction history for a single party
- Header: Party name, type (Customer/Supplier), total balance
- Timeline of all invoices and payments
- Running balance calculation

### SCREEN 8: Settings

**Layout**: Grouped list

- **User Card**: CircleAvatar with initial, company name, email
- **GENERAL Section**:
  - Profile Settings (edit business details)
  - Appearance (theme toggle: Light/Dark/System)
- **LEGAL Section**:
  - Privacy Policy → opens browser
  - Terms & Conditions → opens browser
- **ACCOUNT Section**:
  - Sign Out
  - Delete Account (red, with confirmation dialog)
- **Footer**: "Outstanding Manager v1.0.0"

### SCREEN 9: Login / Register

- Clean, minimal auth flow
- Google Sign-In button
- Email/password fields
- App logo at top
- Coral red primary action button

## Key Visual Features to Highlight

### 1. Pulse AI Assistant (Hero Feature)
- User types: "received 50k from vasant"
- App instantly records a ₹50,000 receipt from "Vasant Traders"
- Auto-allocates to oldest unpaid invoices
- Show the chat-like bottom sheet interface, sparkle icon

### 2. Smart Dashboard
- Real-time financial overview
- Color-coded net outstanding (green = net receivable, red = net payable)
- Side-by-side sales vs purchase cards

### 3. AI Collection Reminders
- Overdue invoice selected → "Send Reminder" button
- AI generates professional, tone-appropriate message
- Friendly for 1-7 days late, firm for 15+ days

### 4. Professional Exports
- PDF reports with branded headers and totals
- Excel spreadsheet exports
- Share via WhatsApp, email, etc.

### 5. Dark Mode
- Full dark mode support
- Dark grey cards (#1E1E1E) on pure black (#121212)
- Same coral red accents pop against dark backgrounds
